개인정보처리방침

최종 업데이트: 2025년 6월

Padlet to Notion Chrome 확장 프로그램(이하 "본 확장")은 사용자의 개인정보를 중요하게 여깁니다. 본 방침은 본 확장이 수집·사용·보관하는 정보에 대해 설명합니다.

1. 수집하는 정보

  • Notion 액세스 토큰: OAuth 인증 후 발급된 토큰을 Chrome 로컬 스토리지에 저장합니다.
  • Padlet 페이지 콘텐츠: 사용자가 저장을 요청한 패들렛 페이지의 제목, 본문, 링크, 이미지 URL을 읽습니다.

2. 수집하지 않는 정보

  • 사용자의 이름, 이메일, 연락처 등 개인 식별 정보를 수집하지 않습니다.
  • 사용 통계, 분석 데이터, 광고 식별자를 수집하지 않습니다.
  • 패들렛 로그인 자격증명을 수집하거나 저장하지 않습니다.

3. 정보의 사용

수집된 정보는 오직 다음 목적으로만 사용됩니다:

  • 사용자가 선택한 패들렛 콘텐츠를 사용자의 Notion 데이터베이스에 저장

어떠한 정보도 제3자에게 판매·공유·전송되지 않습니다.

4. OAuth 및 서버 처리

Notion 계정 연결 시 OAuth 토큰 교환 과정에서 인증 코드가 본 확장의 서버(Vercel 서버리스 함수)를 경유합니다. 서버는 토큰 교환만을 처리하며, 인증 코드 및 토큰을 저장하거나 기록하지 않습니다.

5. 데이터 저장 위치

모든 데이터(Notion 토큰, DB 연결 설정)는 사용자의 기기 내 Chrome 로컬 스토리지에만 저장됩니다. 별도의 외부 서버에 사용자 데이터를 저장하지 않습니다.

6. 데이터 삭제

본 확장을 Chrome에서 제거하면 저장된 모든 데이터가 삭제됩니다. 확장 내 설정 화면에서 개별 연결을 삭제할 수도 있습니다.

7. 문의
